scikit-learn

Python机器学习开发包

  • 简单高效的工具用于数据挖掘和数据分析
  • 任何人可以免费获得，并且在任意场景下使用
  • 基于 NumPy, SciPy, 和 matplotlib 构建
  • 开源, 商业使用许可 - BSD license

分类

确定一个对象属于哪个类别。

应用: 垃圾邮件检测, 图像识别.
算法: 

SVM, nearest neighbors, random forest, ...

实例

回归

预测与一个对象相关联的连续值属性。

应用: 药物反应，股票价格。
算法: 

SVR, ridge regression, Lasso, ...

实例

聚类

将相似对象自动分组到同一个集合.

应用: 客户归类，实验结果分组
算法: 

k-Means, spectral clustering, mean-shift, ...

实例

维数约简

压缩减少随机变量的个数或特征分量的个数.

应用: 可视化, 提高效率
算法: 

PCA, feature selection, non-negative matrix factorization.

实例

模型选择

通过比较、验证来选择合适的模型及模型参数.

目标: 通过参数调节提高准确性
模块: 

grid search, cross validation, metrics.

实例

预处理

特征提取与归一化.

应用: 将原始输入数据比如图像、文本等转换为机器学习算法可操作的一组属性值或特征向量。
模块: 

preprocessing, feature extraction.
